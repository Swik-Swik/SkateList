# SkateList

A web application for managing and viewing skateboarding trick videos.

## Quick Start

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Start the server:**

   ```bash
   npm start
   ```

3. **Open in browser:**

   ```bash
   http://localhost:3000
   ```

## Project Structure

- `index.html` - Main application page
- `scss/` - Stylesheets (SCSS format)
- `js/` - JavaScript files
- `server.js` - Express server
- `videos/` - Video files
- `images/` - Image assets

## Technologies

- Express.js server
- Bootstrap 5 for UI components
- SCSS for styling
- Vanilla JavaScript
